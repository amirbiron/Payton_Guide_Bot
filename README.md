## 🚀 התקנה מהירה

### צעד 1: קבל טוכן מ-BotFather
```
1. פתח @BotFather בטלגרם
2. שלח: /newbot
3. בחר שם ו-username
4. העתק את הטוכן
```

### צעד 2: התקן תלויות
```bash
pip install -r requirements.txt
```

### צעד 3: הגדר טוכן
פתח `bot.py` ושים את הטוכן בשורה 8:
```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
```

### צעד 4: הרץ!
```bash
python bot.py
```

תראה:
```

📚 טוען 20 שיעורים...

```

## 🌐 הרצה 24/7

רוצה שהבוט יעבוד תמיד? אפשרויות:

### 1. Railway (מומלץ)
```bash
1. הירשם ל-railway.app
2. הוסף את הפרויקט
3. הגדר BOT_TOKEN כ-environment variable
4. Deploy!
```

### 2. PythonAnywhere (קל)
```bash
1. הירשם ל-pythonanywhere.com
2. העלה קבצים
3. התקן: pip install -r requirements.txt
4. הרץ: python bot.py
5. הוסף ל-Always-on tasks
```

### 3. VPS (למתקדמים)
```bash
1. שכור VPS
2. התקן Python
3. העלה קבצים
4. הרץ עם screen/tmux
```
